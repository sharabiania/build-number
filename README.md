# Setup Instructions
## Install Dependencies
- Install NodeJS.
- Install MongoDB.
## Install NodeJS Packages
- `npm install express`.
- `npm install mongodb`.
## Run
- Create the database for the first time: `node create_db.js` (one time only).
- Rund the server: `node server.js`.
- Open `index.html` in a browser.
